# Patchwork Update Script
A Patchwork update and launch script for Endless Foundation.

## How to Use

1. Place `update_patchwork.cmd`, `pwsh_update_patchwork.ps1`, and `launch.cmd` in your Godot project's root
2. Run `update_patchwork.cmd` to update patchwork
3. Run `launch.cmd` to launch Godot with the downloaded editor
4. Make sure to merge in this repo's `.gitignore` with your own. 

Note: This script only works on Windows.
